◆ setVsync()

void GameWindow::setVsync ( bool flag)

Setter for if Vsync should be used.

Usage:

THIS FUNCTION MUST ONLY BE CALLED AFTER Renderer::init()! Calling this function before initializing the renderer will lead to a GLFW_NO_CURRENT_CONTEXT error.

This sets up OpenGL to wait for the screen to fully update before swapping the buffers.

Explanation:

Default behaviour for buffer swapping varies by device, so setting this ensures vsync behaviour. If set to false, buffer swapping occurs acyncronouosly as soon as the frame is ready. If set to true, buffer will wait for screen update before swapping.

flag

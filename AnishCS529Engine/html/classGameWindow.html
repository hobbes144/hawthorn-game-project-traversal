void GameWindow::setResizeCallback ( std::function< void(GLFWwindow *, int, int)> callback)

Function that sets the resize callback to the resize callback wrapper.

This is used to setup the callback function that performs the following when the user resizes the window:

  • Update the window's width and height.
  • Update the drawable width and height area in the Renderer class.
Parameters
callbackThe callback function sent by the Renderer, which needs to be registered to glfwSetFramebufferSizeCallback.

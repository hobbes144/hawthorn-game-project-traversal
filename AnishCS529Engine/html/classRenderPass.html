Detailed Description

Render Pass class used to render using a shader.

Usage:

This class is meant to be used to render using a shader. Each shading pass for rendering should be handled using a RenderPass.

Typically after the shader and resources are set, this class should be registered to the RenderGraph, which will handle rendering.

Pre-registration calls:

  • setResource(const std::string& name, void* value)

If FBO functionality is needed,

  • setFBO(std::shared_ptr<FBO> FBO)

Explanation:

The class provides an interface for the Shader class. It stores the required resources to run the specific pass, and applies them when apply() is run.

FBO functionality is supported if needed, and is diabled by default unless the setFBO() call is used.

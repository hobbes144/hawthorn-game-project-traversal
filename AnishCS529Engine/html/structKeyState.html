Detailed Description

KeyState struct holds the state of each key.

The key's current state is handled by storing the previous and current key state

The key can be:

  • pressed (was released, now pressed)
  • released (was pressed, now released)
  • down (is pressed)
  • up (is released)
  • held (was pressed, still pressed)

These states help determine when the state has a shift but also if it's held.

The held keystate is managed by the Input class and isn't an OpenGL feature.
